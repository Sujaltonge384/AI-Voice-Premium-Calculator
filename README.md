# 🧮 AI Voice Premium Calculator

A **next-generation calculator web app** built with **HTML, CSS, and JavaScript**, featuring a premium UI, voice assistant, keyboard support, and full responsiveness across all devices.

---

## 🚀 Live Demo

🔗 (https://sujaltonge384.github.io/AI-Voice-Premium-Calculator/)

---

## ✨ Features

* 💎 **Premium Glassmorphism UI**
* 🌗 **Dark / Light Mode Toggle**
* 🎤 **AI Voice Assistant (Speech Recognition + Speech Output)**
* ⌨️ **Full Keyboard Support**
* ⚡ **Real-time Calculations**
* 📱 **Fully Responsive Design (Mobile + Tablet + Desktop)**
* 🎨 **Smooth Animations & Micro-interactions**
* 💾 **Theme Persistence using LocalStorage**

---

## 🎤 Voice Commands

Try speaking:

* “2 plus 3”
* “10 divided by 2”
* “5 times 6”
* “clear”
* “calculate”

👉 The calculator:

* Converts speech → math
* Solves automatically
* Speaks the result 🔊

---

## ⌨️ Keyboard Controls

| Key       | Action        |
| --------- | ------------- |
| 0–9       | Numbers       |
| + - * / % | Operators     |
| Enter     | Calculate (=) |
| Backspace | Delete        |
| Esc       | Clear         |
| .         | Decimal       |
